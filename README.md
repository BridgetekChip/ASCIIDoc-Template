# ASCIIDoc-Template

Pls open the folder with VS code, which will load the container for ASCIIDoc automatically.    

It may take some time for the first time.  Pls allow "Dev Container" extension to be installed. 

The main document is datasheet.adoc file. You can preview it in VS code.

The html and pdf are generated files so don't edit them.

Most of the ASCIIDoc contents are generated by ChatGPT.  

You may ask any questions to ChatGPT with  "How to do xxx in ASCIIDoc".  

Hopefully, it will save a lot of efforts on documentation.
