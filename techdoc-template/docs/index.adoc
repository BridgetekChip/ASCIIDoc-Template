==  中文段落与排版

这是一段中文示例：我们希望在 *HTML* 预览与 *PDF* 导出中都能正确显示中文、标点符号与换行。

TIP: 在 PDF 中出现“□”方块通常是字体不完整或 PDF 主题未正确配置。

==  Recommended workflow

. Write in `docs/index.adoc`
. Preview in VS Code (AsciiDoc extension)
. Build HTML/PDF via tasks or scripts

==  Getting Started

* Install dependencies using `apt` or your preferred package manager
* Clone the repository with `git clone` and navigate to the workspace
* Open the AsciiDoc file in VS Code with the AsciiDoc extension installed
* Use the preview pane to see real-time rendering of your documentation
* Build HTML and PDF outputs using the provided build scripts or tasks
