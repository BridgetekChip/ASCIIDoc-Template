=== UART 初始化
以下是基于 *C 语言* 的初始化代码示例：

[source,c]
----
#include "mcu_uart.h"

void uart_init(uint32_t baudrate) {
    UART->BAUD = SystemCoreClock / baudrate; // 设置波特率
    UART->CTRL |= UART_ENABLE;               // 使能串口
}
----

== 公式与计算
对于 PWM 频率的计算，公式如下：

[latexmath]
++++
f_{PWM} = \frac{f_{clk}}{(Prescaler + 1) \times (Period + 1)}
++++

use ASCII math, 公式如下: 

:stem: asciimath

[asciimath]
++++
f_(PWM) = f_(clk) / ((Prescaler + 1) * (Period + 1))
++++


== 寄存器描述 32-bit Register Example


=== GPIO_MODE Register (0x00)

==== Register Layout (32-bit)

[cols="1,4,1,6", options="header"]
|===
| Bits | Field | Reset | Description
| 31:16 | RESERVED | 0x0000 | Reserved, must keep default value.
| 15:8  | SPEED    | 0x00   | Output speed selection.
| 7:4   | PULL     | 0x0    | Pull-up / pull-down configuration.
| 3:0   | MODE     | 0x0    | Pin mode selection.
|===

==== Field Definitions

* *MODE* (Bits [3:0])
** `0x0` = Input
** `0x1` = Output
** `0x2` = Alternate Function
** `0x3` = Analog

* *PULL* (Bits [7:4])
** `0x0` = No pull
** `0x1` = Pull-up
** `0x2` = Pull-down

* *SPEED* (Bits [15:8])
** `0x00` = Low
** `0x01` = Medium
** `0x02` = High
** `0x03` = Very High

==== C Access Example

[source,c]
----
#include <stdint.h>

#define GPIO_BASE      (0x40020000UL)
#define GPIO_MODE_OFF  (0x00UL)
#define GPIO_MODE      (*(volatile uint32_t *)(GPIO_BASE + GPIO_MODE_OFF))

// Bit positions / masks
#define GPIO_MODE_MODE_Pos   0
#define GPIO_MODE_MODE_Msk   (0xFUL << GPIO_MODE_MODE_Pos)

#define GPIO_MODE_PULL_Pos   4
#define GPIO_MODE_PULL_Msk   (0xFUL << GPIO_MODE_PULL_Pos)

#define GPIO_MODE_SPEED_Pos  8
#define GPIO_MODE_SPEED_Msk  (0xFFUL << GPIO_MODE_SPEED_Pos)

static inline void gpio_mode_set(uint32_t mode, uint32_t pull, uint32_t speed)
{
    uint32_t reg = GPIO_MODE;

    reg &= ~(GPIO_MODE_MODE_Msk | GPIO_MODE_PULL_Msk | GPIO_MODE_SPEED_Msk);
    reg |= ((mode  & 0xFUL)  << GPIO_MODE_MODE_Pos);
    reg |= ((pull  & 0xFUL)  << GPIO_MODE_PULL_Pos);
    reg |= ((speed & 0xFFUL) << GPIO_MODE_SPEED_Pos);

    GPIO_MODE = reg;
}

static inline uint32_t gpio_mode_get(void)
{
    return GPIO_MODE;
}

----





[discrete]
== Notes

* This register is 32-bit aligned, so offsets increment by 4 bytes (0x00, 0x04, 0x08...).
* Always preserve *RESERVED* bits (read-modify-write) unless the spec says otherwise.
