# Codex Skills

This directory stores version-controlled Codex skills for this repository.

The canonical project copy of the AsciiDoc skill lives at:

- `.codex/skills/asciidoc-datasheet-authoring/`

To install or refresh the user-local copy that Codex auto-discovers, sync it into `~/.codex/skills`:

```bash
mkdir -p ~/.codex/skills
rm -rf ~/.codex/skills/asciidoc-datasheet-authoring
cp -R .codex/skills/asciidoc-datasheet-authoring ~/.codex/skills/
```

Keep the repository copy as the source of truth for version control.
